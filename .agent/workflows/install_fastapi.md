---
description: Install FastAPI dependencies and run the FastAPI app
---

1. Open a terminal in the project root `c:\Users\g.aa.chandrasekaran\Code\fastpoc\python`.
2. (Optional) Create a virtual environment:
   ```
   python -m venv venv
   .\\venv\\Scripts\\activate
   ```
3. Install required packages:
   ```
   pip install fastapi uvicorn
   ```
4. Verify installation:
   ```
   pip list | findstr fastapi
   ```
5. Run the application:
   ```
   py .\\main.py
   ```
   // turbo-all
```
